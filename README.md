## 주요 기능

### 🧑‍🤝‍🧑 워크스페이스

> 원하는 사람들과 워크스페이스를 만들어 **회의**하고 **회의 기록을 관리**해요.

### 🎥 화상 회의

> 워크스페이스 멤버들과 **화상 회의**를 진행할 수 있어요.
